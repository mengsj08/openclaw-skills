# 🦞 OpenClaw Skills — 龙虾养成技能包

> OpenClaw AI Agent 框架的四大核心配置技能，帮助老板快速上手、配置属于自己的 AI 员工。

## 📦 技能包清单

| 技能名称 | 功能描述 | 生成文件 |
|---|---|---|
| `openclaw-beginner-guide` | 入门引导：用「养龙虾」比喻讲清 OpenClaw 三大核心原则 | — |
| `openclaw-soul-collector` | 灵魂配置：通过问答生成 SOUL.md（世界观与行为边界） | `SOUL.md` |
| `openclaw-agents-builder` | 岗位配置：通过问答生成 AGENTS.md（职责与红绿灯权限） | `AGENTS.md` |
| `openclaw-heartbeat-builder` | 心跳配置：通过问答生成 HEARTBEAT.md（自动巡检与日程） | `HEARTBEAT.md` |

## 🚀 使用方法

将所需技能文件夹复制到你的 OpenClaw 技能目录：

```bash
cp -r openclaw-beginner-guide ~/.openclaw/skills/
cp -r openclaw-soul-collector ~/.openclaw/skills/
cp -r openclaw-agents-builder ~/.openclaw/skills/
cp -r openclaw-heartbeat-builder ~/.openclaw/skills/
```

然后在 OpenClaw 中调用对应技能即可开始配置。

## 🧠 核心理念

1. **老板是第一养虾人** — 不要把 AI 配置工作完全委托给员工
2. **把龙虾当实习生** — 需要耐心引导，越用越懂你
3. **龙虾已博览群书** — 只需告诉它你的业务上下文，不需要基础培训

## 📄 配置文件说明

- **SOUL.md** — 龙虾的灵魂：核心真理、行为边界、风格调性
- **AGENTS.md** — 岗位说明书：工作流程、红绿灯权限系统
- **HEARTBEAT.md** — 自动巡检：定时任务、消息监控、日程管理
- **MEMORY.md** — 记忆笔记本（自动生成）：长期偏好与每日日志
